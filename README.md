# Электронный журнал

Веб-приложение для учёта посещаемости, оценок и лабораторных работ.  
Роли: **Преподаватель** и **Студент**.

## Технологии

| Часть | Стек |
|-------|------|
| Backend | Node.js + TypeScript + Express + Prisma |
| Frontend | React 18 + Vite + TypeScript + Ant Design |
| База данных | SQLite (файл `backend/prisma/dev.db`) |
| Авторизация | JWT |
| Файлы | Multer (локально, папка `backend/uploads/`) |

---

## Запуск проекта

### Шаг 1 — Установить Node.js

Скачать: https://nodejs.org (версия 18 или выше)

Проверить установку:
```
node --version
npm --version
```

---

### Шаг 2 — Установить зависимости

Открыть **два терминала** (или два окна PowerShell).

**Терминал 1 — Backend:**
```
cd backend
npm install
```

**Терминал 2 — Frontend:**
```
cd frontend
npm install
```

---

### Шаг 3 — Настроить и запустить базу данных

В терминале с backend:
```
npx prisma generate
npx prisma migrate dev --name init
npm run db:seed
```

После seed в консоли появятся тестовые аккаунты.

---

### Шаг 4 — Запустить

**Терминал 1 (backend):**
```
npm run dev
```
Сервер запустится на http://localhost:3001

**Терминал 2 (frontend):**
```
npm run dev
```
Приложение откроется на http://localhost:5173

## Функционал

### Преподаватель
- Расписание — просмотр и добавление слотов
- Журнал — таблица оценок и посещаемости
  - ЛКМ → выставить оценку (1–10)
  - ПКМ → отсутствие / снять отсутствие
  - СКМ → опоздание / снять опоздание
  - Подсветка строки, столбца и ячейки при наведении
  - Отчисленные студенты — затемнены, новые — синим цветом
  - Автоопределение опоздания при создании занятия
- Программа предмета — создание лабораторных, контрольных, практик
  - Загрузка ТЗ / материалов
  - Командные работы (формирование команд)
  - Дедлайны
- Сдача лабораторных — просмотр файлов, выставление оценок и комментариев

### Студент
- Расписание — актуальное расписание по дням
- Журнал — все оценки и посещаемость, статистика
- Страница предмета — разбивка по типам занятий (лекции, практики, лабы, контрольные)
- Лабораторная — загрузка решения, просмотр комментария и оценки преподавателя

```

├── backend/          — Express API

    └── src/

        ├── pages/    — страницы (student + teacher)

        └── store/    — Zustand (авторизация)
```
